# Shōhai

**AI-Powered Job Posting Arbitrage Assistant**

Upload a job posting screenshot, and Shōhai analyzes it to identify automation opportunities in administrative roles. Get a complete breakdown of the bottlenecks, proposed AI architecture, implementation timeline, and financial arbitrage potential.

## Features

- **Visual Job Analysis** - Upload any job posting screenshot for automated analysis
- **Bottleneck Detection** - Identifies raw administrative pain points (data entry, invoice processing, record updates, etc.)
- **AI Architecture Blueprint** - Three-node pipeline visualization (Trigger → LLM → Action)
- **Custom Implementation Timeline** - Job-specific 3-phase rollout plan generated by Mistral AI
- **Financial Arbitrage Dashboard** - Side-by-side cost comparison between human FTE and AI agent

## Tech Stack

| Layer | Technology |

## Getting Started

### Prerequisites

- Python 3.11+ (for backend)
- Node.js 20+ (for frontend)
- MISTRAL_API_KEY (from [Mistral AI Console](https://console.mistral.ai))

### Installation

```bash
# Clone the repository
git clone https://github.com/yaqoah/shohai.git
cd shohai

# Install Python dependencies
pip install -r requirements.txt

# Install frontend dependencies
npm install
```

### Environment Setup

Create `.env.local` in the project root:

```env
MISTRAL_API_KEY=your_api_key_here
LANGFUSE_PUBLIC_KEY=your_public_key
LANGFUSE_SECRET_KEY=your_secret_key
LANGFUSE_HOST=https://cloud.langfuse.com
PROD_FRONTEND_URL=https://your-domain.vercel.app
```

### Development

Run both frontend and backend locally:

```bash
npm run dev:full
```

Or separately:

```bash
# Backend only (port 8000)
python -m uvicorn api.index:app --reload --port 8000

# Frontend only (port 5173)
npm run dev
```

### Production Build

```bash
npm run build
```

## API Endpoints

### `GET /api/health`
System health check with uptime and environment status.

### `POST /api/analyze`
Analyze a job posting image.

**Request Body:**
```json
{
  "image": "data:image/jpeg;base64,..."
}
```

**Response:**
```json
{
  "job_title": "Accounts Payable Specialist",
  "company_name": "Acme Corp",
  "extracted_job_description": "Manages invoice processing...",
  "detected_bottleneck": "Manual invoice data entry into ERP system.",
  "estimated_monthly_tasks": 2500,
  "inferred_seniority": "Mid",
  "proposed_nodes": [...],
  "calculated_financials": {...},
  "implementation_timeline": [...]
}
```

## Deployment

The project is configured for Vercel with automatic Python serverless function detection.

### Vercel Configuration

```json
// vercel.json
{
  "rewrites": [
    { "source": "/api/(.*)", "destination": "/api/index.py" },
    { "source": "/(.*)", "destination": "/index.html" }
  ]
}
```

Push to `main` branch triggers automatic deployment:

```bash
git push origin main
```
